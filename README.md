# Catálogo Arqueológico de Guatemala 🇬🇹

Este proyecto busca centralizar y visualizar la información de los sitios arqueológicos del país, facilitando el acceso a fichas técnicas y ubicación geográfica. En posteriores fases la idea es complementar con informes, fotografias. 

## Estructura del Proyecto
* `/sitios`: Contiene las fichas técnicas en formato Markdown.
* `mapa.geojson`: Base de datos geoespacial de los sitios.
* `index.html`: Interfaz del mapa interactivo.

## Cómo colaborar
1. Haz un Fork del repositorio.
2. Añade información o corrige datos en los archivos .md.
3. Envía un Pull Request para revisión.
